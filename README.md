# 玉玄道运营绩效驾驶舱

面向玉玄道客户演示的运营绩效驾驶舱与客户 OneID 数据整合系统。系统围绕科杰数据底座能力，展示从多源业务系统接入、DWD 明细整合、DWS 汇总预聚合、ADS 场景集市到经营驾驶舱和客户画像的完整链路。

## 演示内容

- 经营分析驾驶舱：覆盖种植、生产、加工、销售、医疗、养生全产业链经营指标。
- 云诊所运营分析驾驶舱：经营概况、销售分析、客流分析、会员分析四大模块。
- 财务综合分析驾驶舱：资本结构、资金效率、偿还能力、盈利能力与风险事项。
- 运营绩效指标体系：指标口径定义、数据范围、分层开发与更新频率。
- 客户 OneID 整合：编码规则、客户匹配、人工复核、同步更新、画像构建。

## 运行方式

仓库根目录的 `index.html` 是自包含静态演示页，可直接打开，也可由 GitHub Pages 发布。

## 发布

仓库包含 GitHub Pages workflow。推送到 `main` 后会自动发布静态演示站点。
