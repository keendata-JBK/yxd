# 玉玄道智能调度中枢

面向玉玄道客户演示的 AI 经营协同系统。系统基于科杰数据底座，将数据资源、知识资产、指标体系、业务流程和经营场景统一组织为 5 类智能体能力。

## 核心模块

- 经营分析智能体：异常发现、原因解释、建议生成、任务派发。
- 客户运营智能体：OneID 合并、客户画像、客户分层、触达策略。
- 云诊所辅助智能体：客户服务记录、医生辅助、疗程建议、复诊提醒。
- 门店督导智能体：门店异常、区域对比、整改任务、闭环复盘。
- 培训知识智能体：企业知识库、智能问答、培训陪练、组织复用。

## 数据底座

系统展示数据资源盘点与多源数据接入、数据治理体系、统一主数据与指标体系、高质量数据集建设，承接 APP、云诊所、ERP、CRM、OA、门店、会员、财务和知识文档等数据源。
